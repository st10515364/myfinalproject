Basically, this app is a simple Quiz App which I built it using Kotlin code. It consists of 3 
screens; the first screen shows a short brief on how to use the app and where to press. 
The user will click on the start button to begin the hack and myth quiz. The user will then 
be taken to the second screen where the first question will automatically pop up, under 
the question section there are 3 buttons. 2 buttons are close together (Hack and Myth). 
The app checks the users answer and immediately gives them feedback whether it’s 
true or false then the user should press the next button to go to the next question up 
until the last question which is question 5. After the user has completed the questions, 
the app navigates to the last screen (Screen 3) which displays the users score alongwith a message based on the user’s performance.
